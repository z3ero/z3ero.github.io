和-两数的下标(两数求和)

题目

给定整数数组，目标数，求数组中和为目标数的两个数字下标记（注意：每个数字只能用一次）

示例

思路

  • 构建（数字-下标集）字典，遍历数组（用目标数-数组元素），获得结果
  • 考虑到重复的相同元素，我们不是一开始就建立好全部映射，而一边判断，一边建立hash

    代码

复杂度分析

  1. 时间复杂度：第一次遍历建立哈希表， 第二次遍历获取结果，O(n)
  2. 空间复杂度：建立了一个 n 个键的hash表，空间复杂度 O(n)

链表合并(两数相加)

题目

给定两个非空链表，返回一个新链表，新链表每个节点值 = 原链表对应位置的值求和 + 进位。 （注意：所有链表节点值为<10的整数）

样例

思路

  • 先将对应节点相加，组成一个链表
  • 对合并后的链表，逐个进行进位处理

    代码

复杂度分析

  1. 时间：遍历了两个链表 O(n)
  2. 空间：没有开辟新链表 O(n)
