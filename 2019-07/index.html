常用的内建模块

time模块

  1. utc: 世界标准时间，格林尼治时间， 中国utc+8

  2. dst(夏令时)： 天亮早的夏季将时间提前1小时

  3. 时间表示形式
    (1) 时间戳： 以整型或者浮点数表示时间，表示距离 1970.1.1的时间
    (2) 元组形式：9个整型元素，表示不同时间含义 

    (year, month, day, hour, minutes, seconds, weekday(0-6), Julian day(1-366), DST flag(-1 or 0 or 1))
()
DST flag: 0表示正常格式ufc，1表示夏令时，-1表示根据当前日期时间格式自行判定
Julain Day 1年中的第几天

    (3) 字符串： 2019-10-01

  4. 相关函数
    time() : 时间戳
    gmtime(): 时间戳 => UTC元组型，默认参数为当前时间
    localtime(): 时间戳 =》本地元组型，默认参数为当前时间
    mktime(): 本地元组型 =》 时间戳
    asctime(): 本地元组型 =》 字符串型
    ctime(): 时间戳 =》 字符串型（周 月 日 时:分:秒 年）格式
    strftime(): 时间元组以指定格式转为字符串形式

    strptime(): 指定格式的字符串解析为时间元组
    sleep(): 单位 s
    clock(): 返回当前程序时间

    • Unix系统返回从程序运行到当前调用clock的时间
    • windows系统不能记录函数开始运行时间，每次clock都返回距离程序第一次调用clock的时间，第一次调用clock之前的程序不算时间

deque

    OrderedDict

  2. 正常使用dict时，key是无序的，OrderedDict使key有序(按照插入顺序)

    Counter

  3. 简单的计数器，dict的子类

uuid模块

  1. 128bit的全局唯一标识符，通常由32字节的字符串表示，保证时间和空间的唯一性（GUID）
  2. 原理：mac地址+时间戳+命名空间+随机数+伪随机数来保证产生的ID唯一性
  3. 作用：随机字符串，可以当做口令使用，比如订单号之类的
  4. 算法：uuid1~5() 每个算法的种子和算法不同

base64

  1. 概述: 对数据进行编码和解码，只是一种通过查表的编码方法，不能用于加密，不够3位用\x00字节在末尾补足，编码后=个数表示补了几位。
  2. 编码原理：
    (1) 对二进制字符串如 b’hello world!’, 每3个字节一组进行处理
    (2) 对3*8=24bit 重新划分为4组，每组6bit, 将6bit转为10进制数(064)
    (3) 根据(2)中的4个0    64数，查表[A-Z, a-z, 0-9, ‘+’, ‘/‘]获得相应的4个字符，就是这3个字节编码后的字符串
  3. 作用:
    适用于小段内容的编码，如数字证书签名，cookie, 网页中传输的少量二进制数据
  4. 异常
    由于base64编码结果中可能出现 ‘+’, ‘/‘ 字符，会造成url歧义
    因此对于url编码，使用 base64.urlsafe_b64encode(str)，相应的，使用 urlsafe_b64decode() 解码

hashlib

  1. hashlib 模块提供了常见的摘要算法，如MD5, SHA1，SHA256, SHA526
  2. 原理：摘要算法(哈希算法，散列算法)：通过一个函数，把任意长度的数据转为一个长度固定的字符串（通常16进制的字符串表示）
    (1) md5： 最常见的摘要算法，速度快，生成32位16进制字符串
    (2) sha1: 结果是160bit, 40位16进制字符串
    (3) sha256: 64位16进制字符串
    (4) sha512: 128位16进制字符串
  3. hash碰撞：因为摘要算法是将无限多的数据映射到有限的集合中，所以会冲突
  4. 应用：用户密码加密，防止脱裤

hmac

  1. 对于同一条数据，key不同得到不同的摘要值
  2. 原理：HMAC算法，用一个key对数据进行杂凑后，再进行hash, key不同，加密后的结果也不同

itertools

  1. 递增无限迭代器：count(start=0, step=1)

  2. 把传入的序列无限重复：cycle(iterable)

  3. 将指定对象重复指定次数或者无限次：repeat(object[, time])

  4. 有限迭代 chain: 将一组迭代对象串联起来，形成更大的迭代器

  5. groupby: 将迭代器中相邻的重复元素拿出来放在一起

  6. 排列组合
    (1) 排列: itertools.permutations(iterable, length)
    (2) 组合: itertools.combinations(iterable, length)
    (3) 笛卡尔积: itertools.product(iterable, repeat=1)
